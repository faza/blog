Viewing Your SQLite Database with the Command Line

SQLite comes with a handy command-line tool that allows you to interact with your database directly. Here’s a quick guide on how to use it:

  1. Open your terminal and navigate to your project directory.

  2. Start the SQLite CLI by running:

    sqlite3 instance/event_registration.db

  3. Once in the SQLite prompt, you can:

    • List all tables:

      .tables

      Example output:

      sqlite> .tables
event  user

    • View table schema:

      .schema user
.schema event

      Example output:

      sqlite> .schema user
CREATE TABLE user (
        id INTEGER NOT NULL, 
        username VARCHAR(64) NOT NULL, 
        email VARCHAR(120) NOT NULL, 
        password_hash VARCHAR(128) NOT NULL, 
        PRIMARY KEY (id)
);
CREATE UNIQUE INDEX ix_user_email ON user (email);
CREATE UNIQUE INDEX ix_user_username ON user (username);

    • Query data:

      SELECT * FROM user;
SELECT * FROM event;

      Example output:

      sqlite> SELECT * FROM user;
